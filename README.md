# PowerBi-Superstore-Sales-Analysis
Power BI dashboard analyzing Superstore sales performance and profitability

## Project Overview
This project analyzes Superstore sales data using Power BI to identify key business insights such as revenue trends, product performance, regional sales distribution, and profitability.

The dashboard helps decision-makers track key performance indicators (KPIs), monitor sales performance, and identify loss-making products.

---

## Tools & Technologies
- Power BI
- Excel
- Data Visualization
- Business Intelligence

---

## Dataset
The dataset contains information about:

- Orders
- Sales
- Profit
- Category
- Product details
- Region
- Customer segment
- Discounts

---

## Dashboard Features

### Sales Dashboard
Key metrics tracked:

- Total Revenue
- Total Orders
- Units Sold
- Average Order Value

Visualizations include:

- Monthly Sales Trend
- Sales by Category
- Top 10 Products
- Sales by Region Map

---

### Profitability Analysis
This dashboard focuses on profit insights:

- Total Profit
- Profit Margin
- Average Discount

Visualizations include:

- Profit by Category
- Profit vs Sales
- Loss Making Products Table

---

## Key Insights
- Technology category generates the highest revenue.
- Some products generate negative profit due to high discounts.
- Regional sales distribution shows strong performance in multiple global markets.
